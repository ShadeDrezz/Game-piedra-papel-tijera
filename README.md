¡Bienvenido a mi versión del clásico juego Piedra, Papel o Tijeras! Este proyecto fue desarrollado para practicar la manipulación del DOM y la lógica de programación en JavaScript, pasando de una versión de consola a una interfaz web interactiva.

📝 Descripción
Desafía a la computadora en un duelo de ingenio y azar. El juego cuenta con una interfaz visual limpia, un sistema de puntuación en tiempo real y una mecánica de " El Mejor de 3".

🚀 Cómo jugar
Abre el archivo index.html en cualquier navegador moderno.

Elige tu arma: Haz clic en uno de los tres botones (🪨 Piedra, 📃 Papel o ✂️ Tijeras).

Observa el resultado: El juego te dirá qué eligió la PC y quién ganó la ronda de forma instantánea.

Gana el las rondas: El primer jugador en alcanzar 3 victorias será coronado como el ganador definitivo.

Reinicia: Una vez terminado el juego, aparecerá un botón para comenzar una nueva partida.

🛠️ Tecnologías utilizadas
HTML5: Estructura semántica del juego.

CSS3: Diseño moderno con Flexbox, transparencias y efectos de hover.

JavaScript (ES6+): Lógica del juego, generación de números aleatorios y manipulación dinámica de elementos (DOM).

Nota del desarrollador: Este proyecto es parte de mi camino aprendiendo desarrollo web, enfocándome en escribir código limpio, eficiente y fácil de mantener.
